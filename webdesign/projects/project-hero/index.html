Project Hero Blocks Showcase

Introduction

This single-page website project brings together the nine hero blocks you created in the Single-column Hero Blocks and Hero Blocks with Images Tutorials.

In this task you will add a tenth hero block that uses a video as a background. This project also includes menus and a footer.

You can display a finished version of this project by clicking the image below.

Learning Goals

At the end of this Tutorial you will be able to:

  • Use a video as a background to a hero block.
  • Add a tinted overlay in front of the video and behind the hero headings and buttons.
  • Add a still image that appears until the video is loaded and ready-to-play.

Portfolio project folders and files

Your first step is to download the files you need for this project.

  1. If a sub-folder named portfolio does not already exist inside your websites folder, create it now.
  2. Download the following compressed file to your computer:  hero.zip
  3. Uncompress the ZIP file into your websites/portfolio sub-folder as shown below.

This will create a sub-folder named hero inside your portfolio folder.

The folders, sub-folders and files for this ‘Hero’ project will be as shown below.

Working with your project files

Now you can begin to work with the files you have downloaded.

  1. In Visual Studio Code, open the following two downloaded files. The first is the web page; the second, the stylesheet:
    • index.html: You will find this in the main folder of your hero folder.
    • style.css: You will find this in the assets/css sub-folder of the hero folder.

You can close any other files you may have open in VS Code.

Downloading the Background Video

Your next step is to download a free-to-use video from the Pexels website.

  1. Click the link below to open a new web browser tab that offers a video for download.   Girl Writing on Her Notebook
  2. At the top right of the screen, click the Free Download button. This will download the video at a resolution of 1920x1080.
  3. Copy the downloaded video to your portfolio/hero/assets/videos sub-folder.
  4. In File/Windows Explorer, rename the video file as follows:   hero-video.mp4.

You are now ready to work with the downloaded video.

Updating the HTML of the hero block

Follow these steps to embed the video file in your web page.

  1. In VS Code, display the index.html file and scroll down to the end.   In the <div> block named hero-10, you can see three comment lines to guide you in working with the hero block content, overlay and background video.
  2. Under the first comment line of “Background video”, copy-and-paste the following HTML. 
    <video loop muted autoplay poster="assets/img/hero-still.jpg" class="background-video">  
    <source src="assets/videos/hero-video.mp4" type="video/mp4">
</video>
    Here you set the file name and folder location of the video you want to use as a background, along with some other properties and values.
  3. Under the second comment line of “Optional overlay”, copy-and-paste the following HTML. 
    <div class="bg-overlay"></div>
    This allows you to include a tinted overlay, if required, to make the text content easier-to-read against the video background.
  4. Under the third comment line of “Hero block content”, copy-and-paste the following HTML. 
    <div class="background-video-content">
   <h1>EDUCATION FOR LIFE</h1>
   <h2>A brighter career and future.</h2>

   <!-- Buttons start here -->
   <div class="container-btn">
      <div class="item-col-2">
         <a href="#" class="btn-primary btn-soft"><i class="fas fa-graduation-cap"></i> Enrol now</a>   
      </div>   
      <div class="item-col-2">
         <a href="#" class="btn-secondary btn-soft"><i class="fas fa-university"></i> All courses</a>  
      </div>   
   </div>
   <!-- Buttons end here -->
</div>
    This provides a container to hold the text content that appears in front of the background video (and also in front of the optional overlay). It typically includes a heading, sub-heading and one or two buttons.
  5. When finished, save your index.html file.

In your web browser, reload and view your index.html web page on desktop/laptop and mobile screens. The background video should play continuously and the page should look as shown below.

That’s it. You have now completed this project.

Uploading your project to GitHub

All the files you have updated for this project are in the hero sub-folder of your websites/portfolio folder.

So the easiest option is simply to upload the entire hero sub-folder to your account on GitHub.

  1. If you are not already signed in to your GitHub account, sign in now.
  2. On the left of your GitHub home page, click the name of the repository that holds your web pages.
  3. On the next GitHub screen displayed, click the portfolio folder.
  4. GitHub should now display your hero folder.
  5. With the portfolio folder displayed on your GitHub screen, click the Add file button and, from the dropdown list displayed, choose the option Upload files.
  6. In File/Windows Explorer on your computer, display your portfolio folder and then drag-and-drop the hero folder to the GitHub tab in your web browser.
  7. After uploading the hero folder, scroll down to the bottom of the GitHub screen, enter a short message in the Commit changes box, click the Commit changes button, and wait for the upload to complete.

Your updated ‘Hero’ project web page is now published on GitHub at a web address similar to the following, where username is the username you have chosen for your GitHub account:

https://username.github.io/portfolio/hero/index.html

or, simply:

https://username.github.io/portfolio/hero

It may take a few minutes for your uploaded files to appear on GitHub.
