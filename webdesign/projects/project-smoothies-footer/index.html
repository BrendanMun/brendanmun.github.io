Smoothies Project:
Footer

Introduction

In this task you will use template content to add a footer section to the 'Smoothies' project you created earlier and to which you then added an email sign-up form.

You can display a finished version of the project web page on GitHub by clicking the image below.

You can view the alternative version with the centred footer here.

Learning Goals

At the end of this Tutorial you will be able to:

  • Copy selected HTML with text content from a template file to a sample web page.
  • Copy selected CSS from a template file to a stylesheet for a sample web page.
  • Edit the copied HTML and CSS as required.

Working with your two HTML files

You will begin by working with two HTML files:

  • The template file you will copy from and
  • The web page you will copy to.

Start Visual Studio Code.

  1. From your websites\templates folder, open one of the following two HTML files. You can choose whichever footer version you like best.
    • footer-template.html   Link
    -or-
    • footer-template-2.html   Link
  2. From your websites\portfolio\smoothies folder, open your web page named index.html. If you have any other files open in Visual Studio, you may wish to close them. This will help you to focus only on the two HTML files you will be working with.

Copy-and-paste the footer HTML

Your next step is to copy content from the footer template file into your index.html web page.

  1. In the footer template file, select and copy the footer section.
  2. In your index.html web page, scroll down to near the end of the web page, to just before the closing </body> tag.
  3. Press the ENTER key a few times to open up some new lines of blank space.
  4. Paste the copied footer content.
  5. When finished, save your index.html web page.

You can now close both the footer template file and your saved index.html web page.

You could create the footer section of the web page with a pair of opening and closing <div>... </div> tags.

However, the more modern HTML5 way is to user the <footer>... </footer> tags.

Working with your two CSS files

Your next task is to work with two CSS files:

  • The template CSS file you will copy from and
  • The stylesheet you will copy to.

In Visual Studio Code:

  1. From your websites\templates folder, open one of the following two CSS files. Choose whichever one is for the footer template file you choose earlier.
    • footer-template.css
    -or-
    • footer-template-2.css
  2. From your websites\portfolio\smoothies\assets\css folder, open your stylesheet named style.css. If you have any other files open in Visual Studio, you may wish to close them. This will help you to focus only on the two CSS files you will be working with.

Copy-and-paste the footer CSS

Your next step is to copy content from the footer template CSS file into your style.css web page.

  1. In your footer template CSS file, select and copy the footer section.
  2. In your style.css file, and click on the last empty line.
  3. Press the ENTER key a few times to open up some new lines of blank space.
  4. On the new last line, paste the copied footer styles.
  5. When finished, save your style.css stylesheet.

You can now close both the template footer CSS file and your saved style.css stylesheet.

Edit the footer hyperlinks

Your next step is to edit the pasted content and links in your index.html file.

  1. In Visual Studio Code, open your index.html web page.
  2. Scroll down to the footer section you have pasted into your web page.
  3. You do not need to change the email address or social media hyperlinks you have included in your footer.   But you will need to edit the hyperlinks to the three other web pages on your website as follows.  
    • Edit the link to your website Home page as follows: 
        <a href="../../index.html">Home</a>
    • Edit the link to your website Portfolio page as follows: 
        <a href="../index.html">Portfolio</a>
    • Edit the link to your website Privacy page as follows: 
        <a href="../../privacy.html">Privacy</a>
    • And if your footer has a link to your website Contact Form page, edit it as follows: 
        <a href="../../contact/index.html">Contact</a>
    • Save your index.html web page and view it in your web browser.

You are now finished working with this file and you can close it in Visual Studio Code.

Edit the footer background colour

Your final step in working with the footer is to change its background colour so that it matches the rest of your project web page.

  1. In Visual Studio Code, open your style.css stylesheet.
  2. Scroll down to the .container-footer selector. This contains the style for the footer background colour.
  3. Change the colour value from the dark grey of #2c3e50 to the deep pink of #d1274b.   If you are using the footer-template-2.html and footer-template-2.css template files, you will want to change the colour of the .container-footer h5 text from grey (#8d9093) to white (#fff).
  4. When finished, save your style.css stylesheet.

Display your web page in your web browser.

You can display a finished version of the project web page on GitHub by clicking the image below.

You can view the alternative version with the centred footer here.

Upload your project to GitHub

Now your web page with the footer is ready to be uploaded to GitHub.

  1. Upload the index.html web page to the portfolio/smoothies/ folder of your GitHub account.
  2. Upload the style.css stylesheet to the portfolio/smoothies/assets/css/ folder of your GitHub account.

Your 'Smoothies' project web page with the footer form is now published on GitHub at a web address similar to the following, where username is the username you have chosen for your GitHub account:

https://username.github.io/portfolio/smoothies/index.html

or, simply:

https://username.github.io/portfolio/smoothies

It may take a few minutes for your uploaded files to appear on GitHub.
