Project van Gogh

Introduction

This single-page website for about Vincent van Gogh contains an embedded and responsive video from YouTube.

It also includes menus and a footer.

You can display a finished version of this project by clicking the image below.

Learning Goals

At the end of this Tutorial you will be able to:

  • Embed a video from an online streaming platform such as YouTube.
  • Add CSS to make the video responsive to the screen size of the user’s web browser.

Portfolio project folders and files

Your first step is to download the files you need for this project.

  2. Download the following compressed file to your computer:  van-gogh.zip
  3. Uncompress the ZIP file into your websites/portfolio sub-folder as shown below.

The folders, sub-folders and files for this ‘van Gogh’ project will be as shown below.

Working with your project files

Now you can begin to work with the files you have downloaded.

  1. In Visual Studio Code, open the following two downloaded files. The first is the web page; the second, the stylesheet:
    • index.html: You will find this in the main folder of your van-gogh folder.
    • van-gogh.css: You will find this in the assets/css sub-folder of the van-gogh folder.

You can close any other files you may have open in VS Code.

Display the index.html web page in your browser. It should look as shown below.

Embedding the YouTube Video

Your first step is to embed the video from YouTube into your web page.

  1. In VS Code, open the index.html file.
  2. Scroll down to just after the h1 heading.
  3. Click the link below to go a video web address on the YouTube website.   Vincent Van Gogh: The Humble Genius
  4. Under the video, click the option named SHARE.
  5. From the options next displayed, choose Embed, and then click Copy. You can now close the YouTube web browser tab.
  6. Return to VS Code and the index.html file.
  7. Paste the copied YouTube embed code into your web page.
  8. In your web browser, reload and view your index.html we page. It may take a few seconds for the embedded video to load, depending on the speed of your Internet connection.

Making the video responsive

The problem with embedding YouTube videos directly is that they are positioned in an iframe container. You cannot make an iframe responsive. A fixed width and height need to be set for it.

Follow these steps to make the embedded video scale according to the width of the user’s screen.

  1. In your index.html file, wrap the video in a new container as follows: 
    <div class="video-container">
  <iframe src="https://www.youtube.com/embed/klZNNUz4wPQ" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>  
</div>
  2. In VS Code, switch to the van-gogh.css stylesheet.
  3. At the bottom of the stylesheet, copy-and-paste the following. 
    .video-container {
   overflow: hidden;
   position: relative;
   width: 100%;
   margin-bottom: 32px;
}

.video-container::after {
   padding-top: 56.25%;
   display: block;
   content: '';
}

.video-container iframe {
   position: absolute;
   top: 0;
   left: 0;
   width: 100%;
   height: 100%;
}
    Note the padding-top: 56.25% style rule. That is set according to the aspect ratio of the video.   In this example, the video’s aspect ratio is 16:9. Hence the figure of 56.25%, which is the result of 9/16.   If the embedded video is 4:3 however, you would set the padding value to 75%.
  4. In your web browser, reload and view your index.html web page on desktop/laptop and mobile screens.

Uploading your project to GitHub

All the files you have updated for this project are in the van-gogh sub-folder of your websites/portfolio folder.

So the easiest option is simply to upload the entire van-gogh sub-folder to your account on GitHub.

  1. If you are not already signed in to your GitHub account, sign in now.
  2. On the left of your GitHub home page, click the name of the repository that holds your web pages.
  3. On the next GitHub screen displayed, click the portfolio folder.
  4. GitHub should now display your portfolio folder.
  5. With the portfolio folder displayed on your GitHub screen, click the Add file button and, from the dropdown list displayed, choose the option Upload files.
  6. In File/Windows Explorer on your computer, display your portfolio folder and then drag-and-drop the van-gogh folder to the GitHub tab in your web browser.
  7. After uploading the van-gogh folder, scroll down to the bottom of the GitHub screen, enter a short message in the Commit changes box, click the Commit changes button, and wait for the upload to complete.

Your ‘van Gogh’ project web page is now published on GitHub at a web address similar to the following, where username is the username you have chosen for your GitHub account:

https://username.github.io/portfolio/van-gogh/index.html

or, simply:

https://username.github.io/portfolio/van-gogh

It may take a few minutes for your uploaded files to appear on GitHub.
