About String Objects

JavaScript has a built-in object named String. Every time you declare a new string variable JavaScript creates a new instance of the string object. A string object has only one property – its length – but has a very wide range of methods that enable scripts to manipulate the string.

String methods fall into four categories:

  • Formatting methods: You can apply font effects such a bold and italic to a string, and change the case of its characters from lower to upper, and vice versa.
  • String analysis methods: You can examine a string to identify the character located at a specified position in the string, and you can examine a string to see whether it contains a specified character (such as '@') or not.
  • String manipulation methods: You can replace occurrences of a particular character or characters with specified alternatives.
  • Substring extraction methods: You can extract parts of a string from the entire string. In the example of the email address 'joe@bloggs.com', you could extract the domain name of 'bloggs.com'.

Whichever type of method you apply to a string, you must set the string equal to the result of the method operation. The following code declares a variable, assigns a value to it, and then applies a method to the variable that converts its content to uppercase characters.

var fName = "john";
fName = fName.toUpperCase;

The result of the above code is that the variable content is set to 'JOHN'. In contrast, the result of the code below is that the value of the variable is still 'John'.

var fName = "john";
fName.toUpperCase;

Examining String Length

A string's sole property is its length. You can write the length of a string named 'address' to the browser window with the following statement.

document.write(<b>String length is: </b>"+address.length);

String length is a read-only property;it cannot be changed by a script. The length property of a string is automatically updated whenever the value in the string changes. the length method is useful for handling information entered by users to a HTMl form. For example, you can show an error message if the user-entered information is either too short or too long.

String Analysis With the indexOf() Method

Perhaps the most common use of the charAt() is to for check for invalid characters in HTML form entries. For example an email address should be restricted to letters, numbers, periods and an '@' symbol. In the statement below the indexOf method searches through the 'email' string, and returns a value of 10. This is the location of the first occurrence of the '@' symbol within the string.

email = john_smith@anywhere.com;
document.write("<B>@ symbol is at position" </B>"+email.indexOf("@");

Two points to remember about the indexOf method:

  • One instance only: The indexOf method shows only the first occurrence of the relevant character(s) within the string. If you want to find all occurrences, you need to repeat the method on the string.
  • Returns -1 if no instance: What happens when the indexOf() method fails to find any instance of the relevant character(s)? Does it return 'undefined'? No. Does it return '0'. No, because a zero would indicate that it has found the relevant at the first location in the string. It returns a -1.

Searching and Replacing String Content

JavaScript's search() method is similar to the indexOf() method, with one difference: you cannot specify a starting character position with search() like you can with indexOf(). All string analyses involving the search() method begin at the start of the string. If the character or characters or not found by the method, it returns a value of -1. The following code provides a simple example.

var mystring = "ECDL goes from strength to strength ";
var str_location = mystring.search("ECDL");

It returns a value of 0, which the position in the string where 'ECDL' starts. If you searched for the word 'strength' within the string the search() method would return a value of 15 because the first occurrence of this word starts at that location.

The replace() method, as its name suggests, replaces one character or series of characters with an alternative. It replaces only the first occurrence of the searched-for character(s) in the relevant string. Within the method's parenthesis you first type the text to find, then a comma, and then the alternative text. Here is a simple example.

var mystring = "ECDL goes from strength to strength ";
var mystring = mystring.replace("ECDL", "ICDL");
document.write(mystring);
