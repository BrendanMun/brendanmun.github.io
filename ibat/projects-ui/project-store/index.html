Introduction

A number of online retailers offer what are called affiliate programmes. An affiliate programme is an agreement in which a business pays another business or influencer ('the affiliate') a commission for sending traffic and/or sales their way. This can be achieved through web content, social media, or a product integration.

For more information, see:

50+ of the Best Affiliate Programs That Pay the Highest Commission

In this project, you will build a web page that offers links to four books from Amazon.

Before continuing, sign up with the Amazon Associates Program.

Project folders and files

Your first task is to create the folders you need for this project.

  1. Inside your ‘main’ 📁 websites folder, create a sub-folder to hold the files for your store project.   Name the new sub-folder 📁 store (all small letters).
  2. Inside this new folder, create an 📁 assets sub-folder.   And inside this, create a 📁 CSS sub-folder and an 📁 img sub-folder.

✅  All done. You now have created the folder with sub-folders for your store.

Creating your Store page in Lunevedy

Follow the steps below.

  1. On the home page of the Lunevedy app, choose the layout Hero block one.
  2. Choose the following menus and options:
  3. Use the other menus and options to select your preferred colours and layout choices for the hero block and menu
  4. Near the bottom-right of the screen, click the HTML button.
  5. In VS Code, display the store/index.html web page.
  6. Select the placeholder nav and header tags as shown below.

Creating the four-column layout

Now it's time to create a four-column layout to store the text and images for your four books.

Working with the four-column layout in Lunevedy

Follow these steps:

  1. On the Home page of the Lunevedy app, click the Four column option.
  2. On the Upper menu, hide the Upper Block.
  3. On the Buttons menu, display a button for each column.
  4. Style your page so that it is visually consistent with the other pages on your website.
  5. When finished, click the HTML button near the bottom-right of the screen.
  6. Paste in the copied content from Lunevedy into your index.html file.
  7. Update the content of the columns with the words from the text.html file and the downloaded images.
  8. Update the hyperlinks to include your Amazon Associates ID.
  9. Return to the Lunevedy app. at the bottom right of the screen, click the CSS button.
  10. In VS Code, display the custom.css file, and paste in the CSS style rules.
  11. Save your index.html and custom.css files.

Finally:

  • Add a nav to your Store web page similar to that on your Home, Portfolio and Contact web pages.
  • Include a link to your Store in the nav of these web pages
  • Add a folder to your Store web page similar to that on your Home, Portfolio and Contact web pages.

Uploading your project folder to GitHub

After finishing your project, you are now ready to upload it to your account on GitHub.

  1. Open a new tab in your web browser and go to GitHub.com. If you are not already signed in to your GitHub account, sign in now.
  2. On your GitHub home page, click the ‘repo’ that holds your web pages. Its name will look as follows, where username is your chosen username on GitHub.   username.github.io
  3. On the next GitHub screen displayed, click the portfolio folder.
  4. With your store folder displayed on GitHub, click Add file button.
  5. From the dropdown list displayed, choose the option Upload files.
  6. In File Explorer (Windows) or Finder (Apple Mac), drag-and-drop your 📁 store folder to upload it to your repository on GitHub.
  7. Scroll down to the bottom of the GitHub screen, and accept or edit the short message (Add files via upload) in the Commit changes box.
  8. Finally, click the green Commit changes button to upload your files.

Your portfolio project web page is now published on GitHub at a web address similar to the following:

https://username.github.io/store/index.html

or simply:

https://username.github.io/store

It may take a few minutes for your uploaded files to appear on GitHub.
