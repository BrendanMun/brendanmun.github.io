Final Checklist

Link to latest versions of Lunevedy CSS and JS files

On your Home, Portfolio, Store and Contact web pages, ensure your are linking to the latest versions of the Lunevedy files:

In the <head>, the CSS link should be:

<link rel="stylesheet" href="https://lunevedy.com/assets/2.0/css/lunevedy.min.css" />

At the bottom of these four web pages, the JavaScript link should be:

<script src="https://lunevedy.com/assets/js/lunevedy.js"></script>

DO NOT use the same meta tags on different web pages of the same website.

This 'confuses' Google and other search engines.

"When two, or more, of your pages contain similar title tags, it falls upon the search engine to try to determine which page the user really wants to see in their results. What happens is that some of your pages will get preferred ranking status, while others get ignored completely. " Source.
