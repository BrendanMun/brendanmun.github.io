2: Download your custom.css stylesheet file

This stylesheet controls the colours of your Home, Portfolio and Contact web pages.

Download one of these two files to the üìÅ assets/css sub-folder of your 'main' websites folder.

  • custom.css (light theme)
  • custom.css (dark theme)

You may already have a custom.css file in your websites/assets/css sub-folder. So rename this downloaded file to custom-1.css.

When you are happy with the layout, colours and content of your web pages linked to custom-1.css, you can then change this stylesheet's name to custom.css.
