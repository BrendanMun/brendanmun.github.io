Scaffolding with Vite

Follow these steps to scaffold your app structure with Vite.

  1. In a terminal, navigate to where you want Vite to create a folder for your app. For example: 
    C:\> react-stuff\apps
  2. Enter the following command that includes the name you want to call your new app. For example, app-react-tw: 
    npm create-vite@latest app-react-tw --template react
  3. Follow the on-screen instructions to build and launch your new app.

Installing TailwindCSS

Follow these steps:

  1. In a terminal, run this command: 
    npm install -D tailwindcss postcss autoprefixer
  2. Next, generate the configuration files: 
    npx tailwindcss init -p
  3. Update your file as follows: 
    module.exports = {
    content: [
       "./src/**/*.{js,jsx,ts,tsx}", // Adjust this line to include any file types that might contain Tailwind classes
       ],
       theme: {
         extend: {},
       },
    plugins: [],
}
  4. In your app /src folder, replace all the index.css file content with the following: 
    @tailwind base;
@tailwind components;
@tailwind utilities;

Start your app. In a browser, your app‛s default web page should now look as shown below.

To ensure TailwindCSS is working, you can add a TailwindCSS utility class to an element in your App.jsx file. For example:

function App() {
    return (
      <div className="App">
        <h1 className="text-3xl font-bold underline">
          Hello, TailwindCSS!
        </h1>
      </div>
    );
}

Creating a Grid Layout with TailwindCSS

Follow these steps:

  1. Create a new component GridExample.jsx in the /src folder with a simple 3-column layout. 
    import React from 'react';

const GridExample = () => {
    return (
        <section className="grid grid-cols-3 gap-4 p-4">
          <div className="bg-blue-500 p-4 text-white">Column 1</div>
          <div className="bg-red-500 p-4 text-white">Column 2</div>
          <div className="bg-green-500 p-4 text-white">Column 3</div>
        </div>
      );
};
    
export default GridExample;
  2. To make the grid response, update it as follows: 
    <section className="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 gap-4 p-4">
  3. Add this new component to app.jsx: 
    import React from 'react';
import './App.css';
import GridExample from './GridExample';
    
function App() {
      return (
        <div className="App">
          <header className="App-header">
            <p>
              TailwindCSS Grid Example in React
            </p>
            <GridExample />
          </header>
        </div>
      );
    }
    
export default App;
