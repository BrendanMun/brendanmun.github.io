About arrays of objects

Arrays of objects combine the power of arrays with the structure of objects. They are commonly used to store collections of related data, such as:

  • Lists of users
  • Product catalogs
  • Social media posts
  • Transaction records

Each item in the array is an object that follows the same property structure, making it easy to organize and process data.

In JavaScript, we can represent this data as an array where each row becomes an object:

Creating an array of objects

Consider the stylesheet-style table of three products below.

Here's how to represent this table in JavaScript using an array of multiple objects. Copy this code to your workfile:

// Creating an array of objects
const arrProducts = [
    {
        productID: 1001,
        name: "Laptop",
        price: 999.99,
        inStock: true
    },
    {
        productID: 1002,
        name: "Mouse",
        price: 24.99,
        inStock: true
    },
    {
        productID: 1003,
        name: "Keyboard",
        price: 59.99,
        inStock: false
    }
];

Note how:

  • The arrProducts array is declared with const to prevent accidental reassignment
  • The entire dataset is an array with square brackets []
  • Each row from the original table is an object with curly braces {}
  • As in regular arrays, the objects are separated by commas
  • All objects follow the same structure (productID, name, price, inStock)
  • Unlike individual objects, the objects in an array do not have a name

Accessing data from an array of objects

Let's focus on the difference between individual objects and objects within an array:

For individual object:

  • The object has a name, such as objLaptop
  • You access properties using the object name and property name, such as objLaptop.price

However, for objects in an array:

  • The objects don't have individual names
  • To get the laptop's price, enter arrProducts[0].price
  • To get the mouse's price, enter arrProducts[1].price

In summary, you access a particular object in an array by its index position.
