Storing spreadsheet-type data as objects

An object stores related properties about a single entity, like a user's first name, last name, age, and email. Each item in an object is called a property, and has two parts: a key (name or label) and a value.

Objects enable you to work with such structured data of related datasets. For example, instead of assigning the details of a product to different variables, you could assign them to a single object.

Each object has a set of properties, each with a key and a value. The keys are the property names, and the values are the data stored in the properties.

As you can see, the colon : character separate a key from its value. And the comma , character signals the end of each key-value pair.

Naming your objects

Object names follow the usual JavaScript rules: they are case-sensitive and cannot contain spaces or the dash (-) character. Here are two tips:

  • Begin your object names with obj, such as objUser and objProduct.
  • Use singular nouns for object names, such as objProduct rather than objProducts.

Creating an object with {} notation

You can create an object using an object name and curly braces {}. This is called object literal notation.

Copy the following code to your objects.html workfile.

// Declaring and populating an object
const objUser = {
    firstName: "Luis", // string
    lastName: "Martinez", // string
    age: 32, // number
    isActive: true // boolean
};

console.log(objUser);

To prevent your objects from being accidentally overwritten, declare them with const rather than let. You can still modify individual properties later if needed.

Displaying a property's value by its key

You can use the key of a property to display its value. JavaScript gives you two ways to do this.

// === Displaying a property inside an object by its key ===

// Using dot notation
console.log(`User's first name: ${objUser.firstName}`);
console.log(`User's last name:  ${objUser.lastName}`);

// Using square bracket notation
console.log(`User's age: ${objUser["age"]}`);
console.log(`User active? : ${objUser["isActive"]}`);

Working with the properties of an object

You can add a new property (key-value pair) to an object, change the value of a property, and delete a property from an object.

// === Working with the properties of an object ===

// Declaring and populating new object
const objCar = {
    make: "Toyota",
    model: "Camry",
    year: 2023,
};
console.log("Original object"); 
console.log(objCar);

// Adding two new properties (key-value) pairs to the object
objCar.color = "Blue";   
objCar.isElectric = false;  
console.log("Added two new properties"); 
console.log(objCar);

// Deleting a property from an object
delete objCar.isElectric;
console.log("Removed a property"); 
console.log(objCar);

// Updating the value within two properties
objCar.color = "Red";   
objCar.year = 2012;   
console.log("Updated two property values"); 
console.log(objCar);

Testing an object for a property

Does an object contain a particular property? You can use the in operator with the relevant key to find out.

// === Testing that a property exists within an object ===
console.log("make" in objCar); // Outputs: true
console.log("owner" in objCar); // Outputs: false

// General test function to check if an object contains a property
function hasProperty(obj, prop) {
    return prop in obj;
}

console.log(hasProperty(objCar, "model")); // Outputs: true

Looping through the properties of object

Use a for...in loop to iterate over all the properties (keys and values ) of an object.

// === Looping through the properties of an object ===
for (const key in objCar) {
    console.log(`${key}: ${objCar[key]}`);
}

The output of the above code is a series of strings. Each string represents a key-value pair from the object. Unfortunately, the format of this output does not make it usable in the rest of the program.

From objects to arrays

JavaScript offers three methods for extracting object properties to arrays.

Object.keys(objName)

Outputs all the keys in an object to an array.

Object.values(objName)

Outputs all the values in an object to an array.

Object.entries(objName)

Outputs each the key-value pair in an object as array. The result is a array of arrays.

Here's two practical examples of outputting an object's keys to an array of arrays.

// === Outputting an object's keys to an array ===
// === Outputting an object's keys to an array ===
const arrUserKeys = Object.keys(objUser);
console.log(`Keys in objUser: ${arrUserKeys}`);
// Outputs: Keys in objUser: firstName,lastName,age,isActive
    
const arrCarKeys = Object.keys(objCar);
console.log(`Keys in objCars: ${arrCarKeys}`);
// Outputs: Keys in objUser: make,model,year,color

The outputs arrUserKeys and arrCarKeys are real arrays that you can access and work with elsewher ein the program as needed.

Next, here's two examples of outputting an object's values to an array.

// === Outputting an object's values to an array ===
const arrUserValues = Object.values(objUser);
console.log(`Values in objUser: ${arrUserValues}`);
// Outputs: Values in objUser: Luis,Martinez,32,true
        
const arrCarValues = Object.values(objCar);
console.log(`Values in objCars: ${arrCarValues}`);
// Outputs: Values in objUser: Toyota,Camry,2012,Red

Again, arrUserValues and arrCarValues are real arrays you can work with.

Finally, here's an example of outputting an object's key-value pairs to an array.


// === Outputting an object's key-value pairs to an array of arrays ===
    
// Create and populate an object
const objProduct = {
    name: "Laptop",
    price: 999.99,
    inStock: true,
    category: "Electronics"
};

// Get array of key-value pairs
const arrProductProps = Object.entries(objProduct);
console.log(arrProductProps);
// Outputs:
// [
//   ["name", "Laptop"],
//   ["price", 999.99],
//   ["inStock", true],
//   ["category", "Electronics"]
// ]

Your web browser console should look similar to the following.

You can work with arrProductProps as you would any other array.

Creating an empty object

You can use const or let to create an object without entering keys or values. See below.

// === Creating an empty object and later adding keys-values ===

// Create empty object
const objPerson = {};
console.log(objPerson.length === 0); // Outputs: true
                
// Add keys and values to empty object
objPerson.firstName = "Emma";
objPerson.lastName = "Smith";
console.log(objPerson.length === 0); // Outputs: false

console.log(objPerson);
// Outputs: {firstName: "Emma", lastName: "Smith"}

Counting the properties in an object

With strings and arrays, you can use their .length property to count the number of characters and elements they contain.

With objects, however, you need to create a function such as the following to find how many properties are contained inside it.

// === Finding the length (number of properties) in an object ===

function countProperties(obj) {
    return Object.keys(obj).length;
}
    
// Example usage:
const obj1 = { name: "Alice", age: 25, city: "New York" };
const obj2 = {};
    
console.log(countProperties(obj1)); // Outputs: 3
console.log(countProperties(obj2)); // Outputs: 0

Working with nested objects

Objects can contain other objects or arrays as properties, enabling the creation of complex data structures.

// === Working with nested objects ===

const person = {
name: "John",
age: 30,
address: {
city: "New York",
zip: 10001
},
hobbies: ["reading", "cycling"]
};

console.log(person.address.city); // Outputs: New York
console.log(person.hobbies[1]); // Outputs: cycling

Object destructuring

Object destructuring allows you to extract properties from objects and assign them to variables.

// === Object destructuring ===

// Without destructuring
const userFirstName = objUser.firstName;
const userLastName = objUser.lastName;

// With destructuring
const { firstName, lastName } = objUser;
console.log(`User name: ${firstName} ${lastName}`);

// Destructuring with different variable names
const { price: productPrice, name: productName } = objProduct;
console.log(`${productName} costs ${productPrice}`);
