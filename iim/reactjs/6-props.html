Props and data types

Only strings can be passed to components without curly braces {}.

For numeric and boolean values, and for arrays, enclose them in one pair of curly braces {}.

For objects, use two pairs of curly braces {{}}.

As an example, add the following inside the <section> tag to your Main.js component file.

<p>The <b>Text</b> is: {props.str}</p>
<p>The <b>Number</b> is: {props.num}</p>
<p>The <b>Truth</b> is {props.truth}</p>
<p>The <b>Array</b> is: {props.arr[0]}</p>
<p>The <b>username</b> in <b>users</b> object is: {props.obj.name}</p>

And update the <Main /> component inside your Apps.js top-level component as follows.

<Main
   props.num = {100};
   companyName="ABC Corporation"
   str ="Hello"
   num={42}
   truth={true}
   arr={[99]}
   obj={{name: 'Alice'}}
  />

Props, state, and hooks

Props are immutable. Their value cannot be changed.

But what if you need variables that might change later? For example, in response to user actions or server responses?

This is where state and hooks come in.

States are data that is first set (initialized) and later managed by a component. Here’s how you define a function component state:

function ParentComponent() {
   const [name, setName] = React.useState("John")
}

In the example below, the state of name is initialized by calling the React.useState function.

The useState function returns an array with two elements:

  • The first element is the value of the state (in this case, “John”).
  • The second element is a function you call to change the state value.

The square bracket surrounding both name and setName is a destructuring array assignment, meaning we take the return value of the useState function, and directly assign them to name and setName variables.

The useState function is an example of what is called in React a 'hook'. These functions let you use React's state feature in functional components.

The setState() method enqueues all of the updates made to the component state and instructs React to re-render the component and its children with the updated state.
