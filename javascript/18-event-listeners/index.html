At the end of this Lesson you will be able to:

About event listeners

Event listeners are the recommended way to invoke functions within modern JavaScript code. You implement an event listener by attaching the addEventListener() method to a button or other element in a web page.

Event listeners have two main advantages over Event handlers:

  • They are located in the JavaScript code, rather than mixed in with the web page content and HTML tags.
  • They can handle the response within the event listener itself, using an anonymous or arrow function. Or they can call external functions located elsewhere in the code.

Events that call an external function

Sometimes you will want an event to call a function located somewhere else in the code. In the example below, a web page contains a button element with an id of btn_1.

<!-- Button with id and class -->
<button id="btn_1">Click me</button>

You could capture a user click on this button with the addEventlistener() method and an external function named showBtnClick() as follows.

// Get button from web page
const btn_1 = document.getElementById('btn_1');

// Attach external function declaration to listen to button.
// Function is called when btn_1 has 'click' event.
// NOTE: No parenthesis () after function name.
btn_1.addEventListener('click', showBtnClick);

// External function called by 'click' event.
function showBtnClick() {
    console.log(`You clicked the button.`);
}

Responding to an event with an external function is a good choice when the event triggers many lines of code. This makes your JavaScript easier to maintain because the response code is separate from the event capture code.

Events that call an internal function

If the response to a button click or other event requires only one or a few lines of code, you might decide that it is unnecessary to create a separate function to process it.

Instead, you can include an anonymous function with a code block {...} inside the parenthesis () of the event listener. See below.

// Create anonymous function inside 'click' event listener
// NOTE: Anonymous function keyword includes parenthesis ()
btn_1.addEventListener('click', function() {
    console.log(`You clicked the button.`);
});
// } closes anonymous function
// ); closes statement containing listener method

You can shorten the anonymous function by writing it in arrow function syntax. See below.

// Attach 'click' listener arrow function to button
btn_1.addEventListener('click', () => {
    console.log(`You clicked the button.`);
});

You do not actually need to create a JavaScript variable btn_1 for the button in the web page. Below is the most concise version of this complete code sample.

// Get button and attach 'click' listener arrowfunction to it.
document.getElementById('btn_1').addEventListener('click', () => {
    console.log(`You clicked the button.`);
});
