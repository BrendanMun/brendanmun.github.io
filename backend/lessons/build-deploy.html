Building an app with Create React App

The create-react-app script enables you to create a minimised,production-ready version of a ReactJS app.

C:\myapps\todos-app> npm run build

This command will:

  • Bundle your app using Webpack.
  • Optimize and minimize the code.
  • Create a /build directory in your project root containing all the production-ready files.

This /build folder contains the following items:

  • static directory: Contains all the CSS and JS files. Filenames will have a unique hash to ensure browser cache busting.
  • index.html: The main HTML file.
  • manifest.json: Provides metadata used when your web app is added to the homescreen on Android.
  • Other assets like images, icons, etc.

Testing your build version locally

After you have built your app, you can test it locally by running the following command.

npm install -g serve

And then:

serve -s build

Now, you can open your browser and navigate to the provided local address to see your production-ready app running.

The /build directory contains all the files you need to deploy. Depending on your hosting provider or platform (like Vercel, Netlify, or a traditional web host), the deployment steps might vary. Generally, you'll be uploading or pointing to the build directory.

Deploying to GitHub Pages

Follow these steps:

  1. Navigate to your ReactJS project folder and install the gh-pages package. This package will allow you to push the build files to a gh-pages branch of your GitHub repository. 
    npm install gh-pages --save-dev
  2. Open your package.json file and make the following modifications: 
    "homepage": "https://[username].github.io/[repository-name]/"
  3. In the same file, add these two scripts: 
    "scripts": {
    // ... other scripts
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build"
  }
    After these additions, a section of your package.json should look something like this: 
        {
        "name": "my-react-app",
        "version": "0.1.0",
        "homepage": "https://[your-github-username].github.io/[repository-name]/",
        "scripts": {
          "start": "react-scripts start",
          "predeploy": "npm run build",
          "deploy": "gh-pages -d build",
          "build": "react-scripts build",
          // ... other scripts
        },
        // ... rest of package.json
      }
  4. Now, you can deploy your app to GitHub Pages by running: 
    npm run deploy
    This command wil build your React app in production mode (because of the predeploy script) and push the contents of the build directory to the gh-pages branch of your GitHub repository using the gh-pages package.

You can now access your app at the following web address:

https://[your-github-username].github.io/[repository-name]/

Every time you make changes to your app and want to update the live version, you'll need to run npm run deploy again.
