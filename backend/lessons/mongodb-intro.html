About MongoDB and Mongoose

MongoDB Atlas is an integrated suite of data services centered around a cloud NoSQL database.

MongoDB stores data in a BSON (Binary JSON) format. Each database contains one or more collections, which are analogous to tables in relational databases. And each collection contains a group of documents. Each document has a unique _id field, serving as a primary key.

Using Mongoose, you can create a schema for your data and then use that schema to create, read, update, and delete data in your MongoDB database.

Installing Mongoose

Follow the steps below.

  1. Create a new directory to store your app. For example, mongodb-basic-app.
  2. In this new directory, create and Express app as in this Tutorial.
  3. In your app directory, enter the following: 
    npm i mongoose
  4. At the top of your app.js file, enter the second line below: 
    const express = require('express')        
const mongoose = require('mongoose');

You are now ready to connect with a MongoDB database.

For more information, see https://mongoosejs.com/.

Signing up to MongoDB

Follow the steps below.

  1. Go to the MongoDB website at https://www.mongodb.com/ and click the green Try free button ay the top right.
  2. Enter an email address and password or use a Gmail account.
  3. Accept the terms and conditions.
  4. Complete the user questionnaire.
  5. On the Deploy a cloud database screen, choose the free Shared option.
  6. On the next screen, click the Build a database button.
  7. Accept or amend the default values for Cloud Provider & Region.
  8. Accept or amend the default Cluster Name and click the Create Cluster button. This may take a few minutes to complete.
  9. Create a database user Username and Password and click the Create User button..
  10. Enter the permitted IP address(es) for accessing your database and click the Add Entry button.
  11. Finally, at the botton of the screen, click the Finish and Close button.
