Learning Goals

At the end of this Tutorial, you will be able to:

  • Use DB Browser to create a sample database and table.
  • Read data from a table using the various features of the SQL SELECT statement.

About the SELECT statement

The SQL SELECT statement is used to retrieve data from one or more tables in a relational database. The data returned is stored in a result table, called the result-set.

To use the SELECT statement, you need a database with a table.

Creating a new database and table

Follow the steps below to create a new database and table with DB Browser.

  1. In DB Browser, click the New Database tab near the top-left of the screen.
  2. Next, click the Create Table tab.
  3. In the Edit table definition dialog box, type users as the table name.
  4. In the Fields tab, use the Add button to create the four fields with names and properties shown below. As you can see, the DB Browser is creating an SQL statement in the lower part of the dialog box. When finished, click OK.
  5. Click the Execute SQL tab.
  6. In the blank space at the top of the tab, paste in the following.
